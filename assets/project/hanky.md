### Description

The Hanky project is a personal exposition project of mine aimed at provide the necessary of knowledge for an introduction to modern mathematics. A large part of it is the math notes that I wrote during my undergraduate years.

These are the details for **Chänzz's The Hanky project**. You can read about the project on my webpage.

**Click [here](/PDFs/Hanky/hanky.pdf) to download the entire project collection. If you like this book and want to support me, please consider [buying me a coffee](/input/donation.html)!!**
